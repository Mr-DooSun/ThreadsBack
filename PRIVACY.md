# Privacy Policy

Last updated: May 12, 2026

## Overview

Who Unfollowed Me? for Threads & Instagram is a Chrome extension that helps users review their own Threads or Instagram follower/following relationships using an official data export file selected by the user.

The extension is designed to work locally in the browser. It does not upload export files or analysis results to an external server.

## Information Processed

The extension processes files that the user explicitly selects through the browser file picker. These files may include Threads or Instagram follower/following export data, including account usernames, profile links, and relationship metadata contained in the export file.

The extension does not collect passwords, authentication tokens, cookies, payment information, health information, location data, private messages, browsing history, or website content from pages the user visits.

## How Information Is Used

The selected export file is parsed locally in the browser to compare following and followers lists. The extension uses this information to show accounts the user follows that do not appear in the followers export.

The extension may save parsed account usernames, profile links, kept accounts, hidden accounts, selected platform preference, and language preference in Chrome local storage so the user can continue reviewing later.

## Data Sharing

The extension does not sell, rent, transfer, or share user data with third parties.

The extension does not transmit selected export files, parsed account data, or analysis results to any external server.

## Remote Code

The extension does not load or execute remote JavaScript or WebAssembly. All extension code is included in the extension package.

## Data Deletion

Users can clear locally stored analysis data from inside the extension using the clear/reset data action. Users may also remove the extension from Chrome to delete extension-managed local data.

## Third-Party Affiliation

This extension is not affiliated with, endorsed by, or sponsored by Meta, Instagram, or Threads.

## Contact

For questions about this privacy policy, contact the developer through the Chrome Web Store listing or the repository where this policy is published.

# 개인정보처리방침

최종 업데이트: 2026년 5월 12일

## 개요

누가 뒷삭했을까? - Threads/Instagram 맞팔 확인은 사용자가 직접 선택한 공식 Threads 또는 Instagram 데이터 export 파일을 사용해 팔로워/팔로잉 관계를 확인할 수 있게 돕는 Chrome 확장 프로그램입니다.

이 확장 프로그램은 브라우저 안에서 로컬로 동작하도록 설계되어 있습니다. export 파일이나 분석 결과를 외부 서버로 업로드하지 않습니다.

## 처리하는 정보

확장 프로그램은 사용자가 브라우저 파일 선택창에서 직접 선택한 파일만 처리합니다. 이 파일에는 Threads 또는 Instagram 팔로워/팔로잉 export 데이터가 포함될 수 있으며, 계정 username, 프로필 링크, export 파일에 포함된 관계 메타데이터가 포함될 수 있습니다.

확장 프로그램은 비밀번호, 인증 토큰, 쿠키, 결제 정보, 건강 정보, 위치 정보, 개인 메시지, 브라우징 기록, 사용자가 방문한 웹페이지의 콘텐츠를 수집하지 않습니다.

## 정보 사용 방법

선택한 export 파일은 브라우저 안에서 로컬로 파싱되며, 팔로잉 목록과 팔로워 목록을 비교하는 데 사용됩니다. 확장 프로그램은 이 정보를 사용해 사용자가 팔로우하지만 팔로워 export에는 없는 계정을 보여줍니다.

확장 프로그램은 사용자가 나중에 이어서 검토할 수 있도록 파싱된 계정 username, 프로필 링크, 유지 목록, 숨김 목록, 선택한 플랫폼 설정, 언어 설정을 Chrome 로컬 저장소에 저장할 수 있습니다.

## 데이터 공유

확장 프로그램은 사용자 데이터를 판매, 임대, 전송 또는 제3자와 공유하지 않습니다.

확장 프로그램은 사용자가 선택한 export 파일, 파싱된 계정 데이터, 분석 결과를 외부 서버로 전송하지 않습니다.

## 원격 코드

확장 프로그램은 원격 JavaScript 또는 WebAssembly를 로드하거나 실행하지 않습니다. 모든 확장 프로그램 코드는 확장 패키지에 포함되어 있습니다.

## 데이터 삭제

사용자는 확장 프로그램 안의 데이터 초기화 기능을 사용해 로컬에 저장된 분석 데이터를 삭제할 수 있습니다. 또한 Chrome에서 확장 프로그램을 삭제해 확장 프로그램이 관리하는 로컬 데이터를 삭제할 수 있습니다.

## 제3자 서비스와의 관계

이 확장 프로그램은 Meta, Instagram, Threads와 제휴하거나 공식적으로 승인받은 제품이 아닙니다.
