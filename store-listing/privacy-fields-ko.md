# Chrome Web Store Privacy Fields - Korean

## 단일 목적

이 확장 프로그램은 사용자가 직접 선택한 Threads 또는 Instagram 공식 데이터 export 파일에서 팔로잉/팔로워 목록을 비교하고, 내가 팔로우하지만 나를 팔로우하지 않는 계정을 보여줍니다.

## 권한 사용 이유

### storage

분석 결과, 유지 목록, 숨김 목록, 언어 설정, 선택한 플랫폼 설정을 Chrome 로컬 저장소에 저장하기 위해 사용합니다.

### sidePanel

확장 프로그램 UI를 Chrome 사이드패널에서 제공하기 위해 사용합니다.

## 데이터 사용

사용자가 파일 선택창에서 직접 고른 Threads 또는 Instagram export 파일을 처리합니다. 파일은 브라우저 안에서만 파싱되며 외부 서버로 업로드, 전송, 판매, 공유되지 않습니다.

확장 프로그램은 사용자가 나중에 이어서 검토할 수 있도록 파싱된 계정 username, 프로필 링크, 검토 상태, 유지 목록, 숨김 목록을 Chrome 로컬 저장소에 저장할 수 있습니다.

## 원격 코드

이 확장 프로그램은 원격 코드를 로드하거나 실행하지 않습니다.

## 데이터 판매 또는 제3자 제공

권장 답변:

- 사용자 데이터를 판매하지 않음
- 사용자 데이터를 제3자에게 전송하지 않음
- 사용자 데이터를 확장 프로그램의 단일 목적 외 용도로 사용하지 않음
- 사용자 데이터를 신용 평가나 대출 목적에 사용하지 않음
