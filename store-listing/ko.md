# Chrome Web Store Listing - Korean

## 패키지 제목

누가 맞팔 안 했을까? - Threads/Instagram 맞팔 확인

## 패키지 요약

내 데이터 파일만으로 Threads 또는 Instagram에서 맞팔하지 않는 계정을 확인합니다.

## 카테고리

도구

## 설명

누가 맞팔 안 했을까? - Threads/Instagram 맞팔 확인은 사용자가 직접 내려받은 Threads 또는 Instagram 공식 데이터 export 파일을 브라우저 안에서 분석해, 내가 팔로우하지만 나를 팔로우하지 않는 계정을 확인할 수 있게 도와주는 작은 도구입니다.

ZIP 또는 JSON 파일을 선택하면 확장 프로그램이 팔로잉 목록과 팔로워 목록을 비교합니다. 파일은 외부 서버로 업로드되지 않으며, 분석은 사용자의 브라우저 안에서만 처리됩니다. 저장된 분석 결과도 Chrome 로컬 저장소에만 남습니다.

이런 상황에 유용합니다.

- 내가 팔로우하지만 맞팔하지 않는 계정 확인
- 프로필을 직접 열어 수동으로 검토
- 확인한 계정은 검토 완료로 기록
- 저장된 데이터를 원할 때 초기화

사용 방법:

1. Meta Accounts Center의 정보 내보내기 화면을 엽니다.
2. Threads 또는 Instagram 프로필을 선택하고 JSON 형식의 export를 요청합니다.
3. 가능하면 `Followers and following`, `Connections` 또는 전체 정보를 선택합니다.
4. 요청이 준비되면 ZIP 또는 JSON export 파일을 내려받습니다. 링크나 스크린샷이 아니라 파일이 필요합니다.
5. Chrome 사이드패널에서 이 확장 프로그램을 엽니다.
6. export 파일을 선택합니다.
7. 브라우저 안에서 계산된 비교 결과를 확인합니다.

이 확장 프로그램은 공식 export 파일만 사용합니다. Threads 화면을 읽거나, 로그인을 대신하거나, 콘텐츠를 스크래핑하거나, 팔로우/언팔로우를 자동으로 수행하지 않습니다.

이 확장 프로그램은 Meta, Instagram, Threads와 제휴하거나 공식적으로 승인받은 제품이 아닙니다.

전체 릴리즈 기록은 GitHub Releases에서 확인할 수 있습니다:
https://github.com/Mr-DooSun/ThreadsBack/releases
