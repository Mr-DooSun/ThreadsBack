# Chrome Web Store Listing - Japanese

## パッケージタイトル

誰がフォローを外した？ - Threads/Instagramフォロバ確認

## パッケージ概要

自分のデータファイルだけを使って、フォローバックしていないアカウントをブラウザ内で確認します。

## カテゴリ

おすすめ: 仕事効率化

ダッシュボードで仕事効率化を選べない場合は、ツールを使用してください。

## 説明

誰がフォローを外した？ - Threads/Instagramフォロバ確認は、ユーザー自身がダウンロードしたThreadsまたはInstagramの公式データエクスポートをブラウザ内で分析し、自分がフォローしているのに相手からはフォローされていないアカウントを確認できる小さなツールです。

ZIPまたはJSONファイルを選択すると、拡張機能がフォロー中リストとフォロワーリストを比較します。ファイルは外部サーバーへアップロードされず、分析はユーザーのブラウザ内だけで処理されます。保存される分析結果もChromeのローカルストレージにのみ残ります。

こんなときに役立ちます。

- 自分がフォローしているのにフォローバックされていないアカウントを確認
- プロフィールを開いて手動で確認
- 確認したアカウントを確認済みとして記録
- 保存データをいつでも初期化

使い方:

1. Metaアカウントセンターの情報エクスポート画面を開きます。
2. ThreadsまたはInstagramプロフィールを選択し、JSON形式のエクスポートをリクエストします。
3. 可能であれば `Followers and following`、`Connections`、または利用可能なすべての情報を選択します。
4. リクエストの準備ができたら、ZIPまたはJSONエクスポートファイルをダウンロードします。リンクやスクリーンショットではなく、実際のファイルが必要です。
5. Chromeのサイドパネルからこの拡張機能を開きます。
6. エクスポートファイルを選択します。
7. ブラウザ内で計算された比較結果を確認します。

最近の更新:

v1.3.0
- Threads / Instagramデータの自動判定を改善しました。
- 分析に反映する前に、アップロードしたファイルをクリアできるようにしました。
- 確認済みボタンの誤クリックリスクを減らしました。

この拡張機能は公式エクスポートファイルのみを使用します。ThreadsやInstagramの画面を読み取ったり、ログインを代行したり、コンテンツをスクレイピングしたり、フォロー/フォロー解除を自動実行したりしません。

この拡張機能はMeta、Instagram、Threadsと提携しておらず、公式に承認または提供されているものではありません。
